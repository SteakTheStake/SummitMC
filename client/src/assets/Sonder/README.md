# Sonder 1024x Images

This folder contains images for the Sonder 1024x resource pack.

## Required Images

Please upload your 5 Sonder 1024x images to this directory with the following exact filenames:

1. **sonder-tree-wide.png** - Wide shot with tree trunk in center and grass blocks
2. **sonder-tree-looking-up.png** - Tree trunk looking upward with blue sky and realistic bark
3. **sonder-tree-yellow-flowers-1.png** - Tree trunk with yellow flowers around it
4. **sonder-tree-yellow-flowers-2.png** - Another tree view with yellow flowers
5. **sonder-texture-samples.png** - Grid showing texture samples (autumn leaves, dead grass, green grass)

## Where These Images Appear

These images have been integrated into:
- **Homepage Slideshow** (`/client/src/pages/Home.tsx`) - Rotates through all images
- **Gallery Component** (`/client/src/components/sections/gallery.tsx`) - Displays in grid with full-screen viewer

Once you upload these files with the exact filenames above, they will automatically appear on the website!
